# Mr. Strong

An action adventure game built with Unity, playable directly in your browser.

**[Play it on itch.io](https://abdullahhamdan.itch.io/ohio-guy)**

---

## Controls

| Input | Action |
|-------|--------|
| `WASD` / Arrow Keys | Move |
| Left Mouse Button | Fire |
| Right Mouse Button | Fire sword *(requires sword item)* |
| `V` | Use special power *(requires a power-up)* |

---

## About
A game where you need to beat your highscore from last time and you need to shoot down different types of enemies. The game has different type of player, abilties, powers and such. There are plenty it's best you play it and find out!
